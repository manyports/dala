# Список всех страниц приложения DALA

Полный список всех страниц Next.js приложения с указанием URL путей, файлов и описания.

## Публичные страницы

### Главная страница
- **URL:** `/`
- **Файл:** `app/page.tsx`
- **Описание:** Главная страница с Hero секцией, статистикой, лентой проектов и призывом к действию

### Просмотр проектов
- **URL:** `/browse`
- **Файл:** `app/browse/page.tsx`
- **Описание:** Страница поиска и фильтрации проектов по категориям с сортировкой

### Как это работает
- **URL:** `/how-it-works`
- **Файл:** `app/how-it-works/page.tsx`
- **Описание:** Информационная страница с объяснением работы платформы

### Цены
- **URL:** `/pricing`
- **Файл:** `app/pricing/page.tsx`
- **Описание:** Страница с информацией о тарифах и условиях платформы

- **URL:** `/dashboard`
- **Файл:** `app/dashboard/page.tsx`
- **Описание:** Обзор проектов пользователя, статистика, созданные и поддержанные проекты

## Примечания

- Все страницы используют Next.js App Router
- Динамические маршруты обозначены квадратными скобками: `[id]`, `[username]`
- Большинство страниц защищены аутентификацией через NextAuth
- Страницы редактора используют общий layout (`EditorLayout`)
- Публичные страницы доступны без авторизации
